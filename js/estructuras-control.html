Secuencial

Es lo que sucede cuando ponemos una línea de código de bajo de la otra. Se ejecutarán en ese orden.

Condicional - if

La sentencia if ejecutrá el código solamente si la condición es true. Se pueden usar operadores lógicos.

                    
                        if (condition){
                            //Condition was true. Code in this block will run.
                        }

La sentencia else ejecuatrá el código dentro del bloque cuando la condición sea falsa. Es opcional.

                    
                        let currentMoney;
                        let laptopPrice;

                        if (currentMoney >= laptopPrice){
                            //Condition was true. Code in this block will run.
                            console.log("Getting a new laptop!");
                        }
                        else{
                            //Condition was false. Code in this block will run.
                            console.log("Can't afford a new laptop, yet!");
                        }

Operador ternario

condición : Una expresión que se evalúa como true o false.

expr1, expr2: Expresión con valores de algún tipo.

                    
                        condición ? expr1 : expr2

Ejemplo:

                    
                        "La Cuota es de:  " + (isMember ? "$2.00" : "$10.00")

Condicional - Switch

La sentencia switch evalúa una expresión, comparando el valor de esa expresión con una instancia case, y ejecuta declaraciones asociadas a ese case 

                    
                        const expr = 'Papaya';
                        switch (expr) {
                        case 'Naranja':
                            console.log('Naranja elegido');
                            break;
                        case 'Mango':
                            console.log('Mango elegido');
                            break;
                        case 'Papaya':
                            console.log('Papaya elegido');
                            break;
                        default:
                            console.log('No se eligió ninguna opción');

Repetitiva - for

El for necesita tres partes para iterar:

  • Contador: Una variable que es inicializada con un número que va a contar la cantidad de iteraciones.
  • Condición: Expresión que usar los operadores de comparación para causar que se detenga el lazo cuando la condición valga true
  • Incremento: Se usa para incrementar el contador
                    
                        for (let i = 0; i < 10; i++) {
                            console.log(i);
                        }

Repetitiva - While

el ciclo while ejecuta un bloque de código mientras la condición sea true

                    
                        let i = 0;
                        while (i < 10) {
                            console.log(i);
                            i++;
                        }
