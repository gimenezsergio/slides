Textos (string)

Es una cadena de tetxo que está entre comillas, pueden ser dobles o simples

Concatenar

                    
                        let cadena1 = 'Este es un texto'
                        let cadena2 = "Otro texto"
                    
                
                    
                        let cadena1 = 'Hola'
                        let cadena2 = "Luis"
                        console.log(cadena1 + cadena2)
                        console.log(cadena1 + ' ' + cadena2)

Template literals

                    
                    let myString1 = "Hola";

                    `${myString1} ${myString2}!`

Booleans

Solo pueden tener dos valores: true o false. En determinados contextos sirve para tomar desiciones. 

                    
                        let producto_entregado = true
                        console.log(producto_entregado)

Objetos

Los objetos nos sirven para representar la realidad. Tienen características y son capaces de realizar acciones.

    
        let usuario = {     
            nombre: "Juan",
            apellido: "Mendez"
            edad: 30,  
            email: "juan@gmail.com",
            password: "miclave",
            nombreCompleto : function() {
                return this.nombre + " " + this.apellido;
                }    
            };

            console.log(usuario.nombre)

Arrays

Los arrays se usan para guardar colecciones de datos ordenadas

                    
                        const fruits = ["Banana", "Naranja", "Manzana", "Pera"];
                        let fLen = fruits.length;

                        for (let i = 0; i < fLen; i++) {
                            console.log(fruits[i])
                        }

Funciones

                        let saludar = function(){
