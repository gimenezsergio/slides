Javascript - Manejo de errores

Los errores van a suceder: del programador, de los datos que ingrese el usuario, que se corte inesperadamente la conexión de internet, etc. Pero te podés adelantar capturandolos.

  • Secuencial
  • Condicional
  • Repetitiva

                    
                        try {
                            miFuncion("Bienvenido!");
                          }
                          catch(err) {
                            console.log(err.message);
                          }

La sentencia throw

Envía un mensaje personalizado

                    
                        try {
                            miFuncion("Bienvenido!");
                          }
                          catch(err) {
                            console.log(err.message);
                            throw "empty";
                          }
